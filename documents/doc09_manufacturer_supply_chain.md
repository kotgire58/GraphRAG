# Pharmaceutical Manufacturer Registry and Supply Chain Data

## Pfizer Inc.
- Manufacturer ID: MFG-001
- Headquarters: New York, New York
- Type: Multinational pharmaceutical corporation
- Revenue (2023): $58.5 billion
- Key drugs manufactured: Fluconazole (Diflucan), Atorvastatin (Lipitor original), Amlodipine (Norvasc), Sildenafil (Viagra, Revatio), Phenytoin (Dilantin), Spironolactone (Aldactone), Cyclosporine (partial)
- Manufacturing facilities: Groton Connecticut, Kalamazoo Michigan, McPherson Kansas, Freiburg Germany, Dalian China
- Active pharmaceutical ingredient sourcing: Significant API sourcing from India and China
- Fluconazole API source: Primarily synthesized at Groton facility; some API from Aurobindo Pharma India
- Generic competition: Diflucan went off-patent 2004; significant generic competition

## Supply Chain Vulnerability Analysis

### Metformin Supply Chain
- Primary API source: India (approximately 80 percent of US metformin API from Indian manufacturers)
- Key Indian API manufacturers: Aurobindo Pharma, Sun Pharma, Macleods Pharmaceuticals
- 2019 shortage event: NDMA contamination found in some metformin products; FDA investigation; voluntary recalls
- NDMA source: Degradation of metformin under certain storage conditions
- Impact of 2019 shortage: Temporary supply disruption; FDA established acceptable intake limits; most products cleared
- Current status: Supply stable; ongoing NDMA monitoring required

### Tacrolimus Supply Chain
- Originator manufacturer: Astellas Pharma (Japan)
- Generic manufacturers: Sandoz (Novartis), Dr. Reddy's Laboratories, Mylan, Accord Healthcare
- API source: Complex fermentation-derived product; limited number of API manufacturers globally
- Critical supply consideration: Narrow therapeutic index means supply interruptions are dangerous; patients cannot simply switch brands without physician oversight
- FDA guidance: Tacrolimus is on FDA's list of drugs where automatic substitution is not recommended without physician authorization

### Fluconazole Supply Chain
- Originator: Pfizer (Diflucan, patent expired 2004)
- Generic manufacturers: Teva, Aurobindo, Mylan, Sandoz, Apotex
- API source: Chemical synthesis; multiple manufacturers in India and China
- Supply status: Generally stable; multiple generic manufacturers provide resilient supply
- Quality considerations: Bioequivalence established for all FDA-approved generics

### Warfarin Supply Chain
- Originator: Bristol-Myers Squibb (Coumadin)
- Generic manufacturers: Taro Pharmaceutical, Mylan, Sandoz
- Supply history: Localized shortages during COVID-19 pandemic (2020 to 2021)
- Narrow therapeutic index consideration: Like tacrolimus, warfarin brand-to-generic switches require INR monitoring

### Insulin Glargine Supply Chain
- Originator: Sanofi (Lantus)
- Biosimilars approved: Basaglar (Eli Lilly), Semglee (Mylan/Biocon), Rezvoglar (Eli Lilly)
- Manufacturing complexity: Biologics require specialized facilities; higher barrier to generic entry than small molecules
- Cold chain requirement: All insulin products require refrigerated supply chain; supply disruption from temperature excursions possible
- Price reduction: Biosimilar entry reduced insulin glargine price approximately 65 to 85 percent in some markets
